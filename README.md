# NaturalLanguageProcessing
These are the programming assignments done for the NLP course under professor grishman, spring 2019.

## Assignment 1:

- Implement and train a part-of-speech tagger using the Viterbi algorithm and Hidden markov models. (Java).
- Handled Unknown words using Hapax Legomenan.

## Assignment 2:

- Built an end to end Named Entity recognizer with Maximum Entropy Classifier. (Java)

## Assignment 3:

- Enhancing the Named Entity tagger using word embeddings and binarization with improving performance with k-means clustering.(Java)
